Strona zawierająca informacje o lidze piłkarskiej "A-klasa Przemyśl".
Użytkownicy niezalogowani mogą przeglądać tabelę, mecze, zawodników oraz transfery.
Użytkownicy zalogowani mają także możliwość dodawania, usuwania oraz edytowania drużyn, zawodników, meczy oraz transferów.
Zalogowani użytkownicy posiadający uprawnienia administratora mogą zarządzać kontami innych użytkowników oraz posiadają wszystkie takie same uprawnienia jak oni.

Tabela aktualizuje się na bieżąco po wprowadzeniu meczu do bazy danych. Analogicznie dzieje się w przypadku dodania transferu, zawodnikowi automatycznie zmienia się drużyna, w której gra.
